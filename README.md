# Desafio Sistema Banco P-O

Este repositório contém a implementação de um sistema de conta bancária desenvolvido em Java, aplicando os pilares da Programação Orientada a Objetos (POO), como encapsulamento e associação de classes.

## 📌 Funcionalidades Mapeadas

- **Gerenciamento de Titular:** Cadastro, atualização e exibição linear de dados de contato (Nome, Endereço, Telefone e E-mail), protegendo informações sensíveis como CPF e ID.
- **Operações Bancárias:** Controle de saldo, agência e número da conta.
- **Segurança:** Sistema preparado para validação de senhas em operações críticas de saque e depósito.

## 🚀 Como Executar o Projeto

1. Certifique-se de ter o **JDK 11 ou superior** instalado em sua máquina.
2. Clone o repositório:
   ```bash
   git clone https://github.com/LucasCampMat/DesafioSistemaBancoP-O.git
   ```
3. Abra o projeto no **IntelliJ IDEA**.
4. Execute a classe `Main` localizada em `src/main/java/org/example/Main.java`.
